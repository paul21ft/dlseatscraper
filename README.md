# dlseatscraper
Python and other scripts for automatically reading seat information from Delta's online website

Current Operation:

Install python 2.7.9 (other versions not tested, Python 3 incompatible)
https://www.python.org/downloads/

When installing, in the options, enable adding python to the PATH

Keep the __init__.py with the main script file in the same folder

Read the top of the python class to test current version
